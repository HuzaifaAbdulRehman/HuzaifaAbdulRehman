## Selected Work

### [Electrolux EMS](https://github.com/HuzaifaAbdulRehman/Electrolux-EMS)

An electricity distribution management system for customer billing, power usage, and service requests. Built with typed validation, database-backed authentication, and a modern full-stack architecture.

`Next.js` `TypeScript` `MySQL` `Drizzle ORM` `NextAuth`

### [Driver Drowsiness Detection](https://github.com/HuzaifaAbdulRehman/driver-drowsiness-detection)

A real-time safety system combining a fine-tuned MobileNetV2 eye-state classifier with MediaPipe facial landmarks. The project reports 97.30% accuracy on the MRL Eye dataset.

`Python` `TensorFlow` `MediaPipe` `OpenCV` `Streamlit`

### [Dijkstra + ML Routing](https://github.com/HuzaifaAbdulRehman/dijkstra-ml-routing-optimization)

Route planning that combines a custom Dijkstra implementation, engineered road features, and XGBoost on real OpenStreetMap networks.

`Python` `NetworkX` `OSMnx` `scikit-learn` `XGBoost`

### [FAST Academic Hub](https://github.com/HuzaifaAbdulRehman/fast-academic-hub)

An offline-first attendance planner that calculates course attendance in real time and models planned absences as an installable, responsive PWA.

`React` `Vite` `Tailwind CSS` `PWA`
